# 📂 Smart File Organizer

A Python script that automatically organizes a messy directory by categorizing files into specific subfolders (Pictures, Videos, Documents, etc.) based on their extensions within seconds.

## 🚀 Features
* Reads the path of any given directory.
* Analyzes file extensions (e.g., `.jpg`, `.pdf`, `.mp4`).
* Automatically creates categorized subfolders if they don't already exist.
* Moves files into their respective folders to keep your workspace clean.
* Safely moves unlisted file types into an "Others" folder.

## 🛠️ How to Use
1. Make sure Python is installed on your computer.
2. Run the script in your terminal or IDE.
3. Enter the full path of the folder you want to organize (e.g., `C:\Users\Username\Downloads`) when prompted.
4. Watch your files get instantly sorted into their new homes!
